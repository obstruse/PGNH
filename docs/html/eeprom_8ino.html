Polargraph Server for ESP32 based microcontroller boards. Written by Sandy Noble Released under GNU License version 3. http://www.polargraph.co.uk https://github.com/euphy/polargraph_server_polarshield_esp32

EEPROM.

This is one of the core files for the polargraph server program.

This doesn't actually use EEPROM in ESP32, but it's the same pattern and interface as other versions of Polargraph so I'm keeping the name.

This uses Preferences which is an ESP32 abstraction library to put and get value into non-volatile storage based on keys.

But mostly it contains the loadMachineSpecFromEeprom method, which is used to retrieve the machines saved values when it restarts, or whenever a value is written to the EEPROM.
